# Demo comic

Add your own images here:

- **cover.png** — cover image (e.g. 400×600 or any 2:3 aspect)
- **pages/** — one PNG per page:
  - 01.png
  - 02.png
  - 03.png
  - 04.png
  - 05.png

Then the rack will show the cover; opening it uses StPageFlip for swipe-to-turn.

If your pages use a different aspect ratio (e.g. 3:4), set `aspect` in `assets/reader.js` (e.g. `4/3`).
